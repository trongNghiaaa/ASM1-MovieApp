# ASM1 - Movie App

Build an Movie App with RESTful API

## Instructions for install and running the code:

### In Client:

```c
> cd FrontEnd
```

```c
> npm install
```

```c
> npm start
```

### In Server

```c
> cd Backend
```

```c
> npm install
```

```c
> npm start
```

## Technology used :

-   Library : ReactJS , Express JS 
-   Database : MonggoDB
